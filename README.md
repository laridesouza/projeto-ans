# Projeto ANS - Análise de Internações Hospitalares

## Objetivo
Investigar se internações de urgência custam mais do que internações eletivas e como esse padrão varia conforme perfil do paciente e tipo de plano de saúde.

## Dataset
Dados públicos da ANS (Agência Nacional de Saúde Suplementar), contendo registros de procedimentos hospitalares realizados por operadoras de planos de saúde no Brasil, segmentados por UF, mês e ano de competência.
Disponível em https://dados.gov.br/dados/conjuntos-dados/procedimentos-hospitalares-por-uf

## Tecnologias Utilizadas
